
### Installation

App requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and start the server.

```sh
$ cd server
$ npm install
$ npm start
```

Visit http://localhost:3000 to check if server started successfully.


### Endpoints

 - /login - GET
 - /organizations - GET
 - /organization/{id} - GET
 - /organization/{id}/users - GET
